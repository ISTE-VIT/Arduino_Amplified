# Happy Birthday Song - Arduino

## About the Project

Happy Birthday Song using Arduino and Piezo Buzzer.

---

## Project Components

1. Microcontroller - Processes all data from the sensor
2. Piezo Buzzer - Produces sound of a tone at a given frequency
