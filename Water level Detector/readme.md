# Required Objects
- UNO Board (Fully compatible with Arduino UNO rev.3) x 1
- Water Sensor x 1
- Jumpers
- USB Cable x 1
- PC x 1

# Pin Layout
“S” stand for signal input
“+” stand for power supply
“-” stand for GND
