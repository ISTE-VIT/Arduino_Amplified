# RGB LED

## About the Project

To control multi-colour LEDs using analog outputs of Arduino.

---

## Project Components

1. Arduino Uno R3 ×1
2. Breadboard ×1
3. Connecting Wires
4. Resistors (200 Ω) ×6
5. RGB LEDs ×2

---

## Programming Language

C++
